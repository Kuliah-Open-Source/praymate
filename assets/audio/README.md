# Audio Adzan

## Cara Menambahkan File Audio Adzan

1. Download file audio adzan pendek (MP3, maksimal 30 detik)
2. Rename file menjadi: `adhan_short.mp3`
3. Letakkan file di folder ini: `assets/audio/adhan_short.mp3`

## Rekomendasi

- Format: MP3
- Durasi: 15-30 detik
- Ukuran: < 500KB
- Bitrate: 128kbps

## Sumber Audio Gratis

- https://freesound.org (search: "adhan" atau "azan")
- https://pixabay.com/sound-effects/search/adhan/
- Rekam sendiri atau gunakan audio bebas royalti

## Catatan

Jika file tidak ditemukan, aplikasi akan tetap berfungsi tanpa suara adzan.
Notifikasi browser tetap akan muncul.
